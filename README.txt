INF01124 - Classificação e Pesquisa de Dados - UFRGS

Aplicação em Python que carrega dados de arquivos .csv, contendo informações de jogadores da FIFA, avaliações de usuários e anotações em texto livre (tags). Após, o programa permite que o usuário realize consultas a esses dados, com diferentes graus de especificidade.
